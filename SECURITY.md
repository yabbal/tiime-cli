# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Do not open a public issue.** Instead, please use [GitHub's private vulnerability reporting](https://github.com/yabbal/tiime/security/advisories/new).

You can expect an initial response within 48 hours.

## Supported Versions

Only the latest version of `tiime-cli` and `tiime-sdk` are supported with security updates.

## Scope

This project is an unofficial, experimental tool. It handles Tiime API credentials locally (stored in `~/.config/tiime/auth.json`). Please be mindful of your credential security.
