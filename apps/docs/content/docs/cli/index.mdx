Le CLI Tiime offre un ensemble complet de commandes pour gérer votre comptabilité depuis le terminal. Chaque commande retourne du JSON par défaut, ce qui les rend facilement combinables avec `jq`, des scripts ou des agents IA.

## Résumé rapide

```bash title="Terminal"
tiime status    # Résumé : soldes, factures en attente, transactions non imputées
```

## Formats de sortie

Toutes les commandes de listing supportent le flag `--format` :

| Format | Description | Usage typique |
|--------|-------------|---------------|
| `json` | JSON structuré (défaut) | Scripts, `jq`, agents IA, piping |
| `table` | Tableau ASCII lisible | Lecture rapide dans le terminal |
| `csv` | Valeurs séparées par des virgules | Export vers Excel, Google Sheets |

```bash title="Terminal"
tiime invoices list --format json    # Défaut — JSON structuré
tiime invoices list --format table   # Tableau lisible
tiime invoices list --format csv     # Export CSV
```

## Flag global `--format`

Le flag `--format` est disponible sur toutes les commandes qui retournent des données :
`list`, `get`, `balance`, `accounts`, `transactions`, `unimputed`, `categories`, `status`, etc.

<Callout type="info" title="Astuce : combiner avec jq">
  La sortie JSON par défaut se combine parfaitement avec [jq](https://jqlang.github.io/jq/) pour filtrer, transformer et extraire des données.

  ```bash title="Terminal"
  tiime invoices list --status paid --all | jq '[.[].total_including_taxes] | add'
  ```
</Callout>
