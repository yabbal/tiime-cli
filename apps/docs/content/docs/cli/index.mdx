Le CLI Tiime offre 13 commandes pour gerer votre comptabilite depuis le terminal.

```bash title="Terminal"
npm install -g tiime-cli
```

## Demarrage rapide

```bash title="Terminal"
tiime auth login           # Se connecter
tiime company use --id ID  # Selectionner l'entreprise
tiime status               # Resume : soldes, factures, transactions
```

## Formats de sortie

Toutes les commandes de listing supportent `--format` :

| Format | Description | Usage typique |
|--------|-------------|---------------|
| `json` | JSON structure (defaut) | Scripts, `jq`, agents IA |
| `table` | Tableau ASCII lisible | Lecture rapide au terminal |
| `csv` | Valeurs separees par virgules | Export Excel, Google Sheets |

```bash title="Terminal"
tiime invoices list --format table
tiime clients list --format csv > clients.csv
```

<Callout type="info" title="Combiner avec jq">
  ```bash title="Terminal"
  tiime invoices list --status paid --all | jq '[.[].total_including_taxes] | add'
  ```
</Callout>
