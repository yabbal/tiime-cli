## Créer et envoyer une facture

<div className="rounded-lg border bg-fd-card p-4 space-y-4 text-sm">
  <div className="flex gap-3">
    <span className="font-semibold text-fd-primary shrink-0">Vous :</span>
    <span>Crée la facture de mars pour Nextera. 22 jours à 600€/jour en développement.</span>
  </div>
  <div className="flex gap-3">
    <span className="font-semibold text-fd-muted-foreground shrink-0">Claude :</span>
    <div className="space-y-2">
      <span>Je cherche le client Nextera et je prévisualise la facture.</span>

      ```bash title="Commandes exécutées"
      tiime clients search --query "Nextera"
      tiime invoices create \
        --client-id 234 \
        --description "Développement - Mars 2026" \
        --quantity 22 --unit-price 600 --unit day \
        --vat normal --dry-run
      ```

      <span>Voici ce qui serait créé :</span>
      <ul className="list-disc pl-4 space-y-1">
        <li>**Client** : Nextera (ID 234)</li>
        <li>**Description** : Développement - Mars 2026</li>
        <li>**22 jours × 600 €** = 13 200 € HT</li>
        <li>**TVA 20%** = 2 640 €</li>
        <li>**Total TTC** = 15 840 €</li>
      </ul>
      <span>Je crée la facture ?</span>
    </div>
  </div>
  <div className="flex gap-3">
    <span className="font-semibold text-fd-primary shrink-0">Vous :</span>
    <span>Oui, et envoie-la à compta@nextera.fr.</span>
  </div>
  <div className="flex gap-3">
    <span className="font-semibold text-fd-muted-foreground shrink-0">Claude :</span>
    <div className="space-y-2">
      ```bash title="Commandes exécutées"
      tiime invoices create \
        --client-id 234 \
        --description "Développement - Mars 2026" \
        --quantity 22 --unit-price 600 --unit day \
        --vat normal
      tiime invoices send --id 156 --email compta@nextera.fr
      ```

      <span>Facture **#156** créée et envoyée à compta@nextera.fr.</span>
    </div>
  </div>
</div>
