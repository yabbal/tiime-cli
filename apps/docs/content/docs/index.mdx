Tiime est un ensemble d'outils TypeScript pour la comptabilite [Tiime](https://www.tiime.fr) :

- **`tiime-sdk`** — SDK pour integrer Tiime dans vos applications
- **`tiime-cli`** — CLI pour piloter votre compta depuis le terminal

<Callout type="warn" title="Projet non officiel">
  Ce projet est personnel et experimental. Il n'est pas affilie a, ni approuve par Tiime.
</Callout>

## Installation

<Tabs items={["SDK TypeScript", "CLI"]}>
  <Tab value="SDK TypeScript">
    ```bash title="Terminal"
    npm install tiime-sdk
    ```
  </Tab>
  <Tab value="CLI">
    <Tabs items={["npm", "Homebrew", "Sources"]}>
      <Tab value="npm">
        ```bash title="Terminal"
        npm install -g tiime-cli
        ```
      </Tab>
      <Tab value="Homebrew">
        ```bash title="Terminal"
        brew tap yabbal/tap && brew install tiime
        ```
      </Tab>
      <Tab value="Sources">
        ```bash title="Terminal"
        git clone https://github.com/yabbal/tiime.git
        cd tiime && pnpm install && pnpm build
        pnpm link --global
        ```
      </Tab>
    </Tabs>
  </Tab>
</Tabs>

## Demarrage rapide

<Tabs items={["SDK TypeScript", "CLI"]}>
  <Tab value="SDK TypeScript">
    <Steps>
      <Step>
        ### Configurer l'authentification

        ```bash title="Terminal"
        export TIIME_EMAIL=vous@example.com
        export TIIME_PASSWORD=votre-mot-de-passe
        export TIIME_COMPANY_ID=12345
        ```
      </Step>

      <Step>
        ### Utiliser

        ```ts title="app.ts"
        import { TiimeClient } from "tiime-sdk";

        const client = new TiimeClient();

        const invoices = await client.invoices.list({ status: "paid" });
        const balances = await client.bankAccounts.balance();
        ```
      </Step>
    </Steps>
  </Tab>
  <Tab value="CLI">
    <Steps>
      <Step>
        ### Se connecter

        ```bash title="Terminal"
        tiime auth login
        tiime company use --id 12345
        ```
      </Step>

      <Step>
        ### Utiliser

        ```bash title="Terminal"
        tiime status
        tiime invoices list --status paid --format table
        tiime bank balance
        ```
      </Step>
    </Steps>
  </Tab>
</Tabs>

## Configuration CLI

### Autocompletion shell

<Tabs items={["zsh", "bash", "fish"]}>
  <Tab value="zsh">
    ```bash title="~/.zshrc"
    eval "$(tiime completion --shell zsh)"
    ```
  </Tab>
  <Tab value="bash">
    ```bash title="~/.bashrc"
    eval "$(tiime completion --shell bash)"
    ```
  </Tab>
  <Tab value="fish">
    ```bash title="Terminal"
    tiime completion --shell fish | source
    ```
  </Tab>
</Tabs>

### Langue

L'aide s'affiche dans la langue de votre systeme. Pour forcer :

```bash title="Terminal"
TIIME_LANG=fr tiime --help   # Francais
TIIME_LANG=en tiime --help   # English
```
