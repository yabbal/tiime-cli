Le SDK TypeScript permet d'intégrer Tiime directement dans vos applications Node.js. Il expose les mêmes données que le CLI sous forme d'API programmatique.

## Installation

<Tabs items={["npm", "pnpm"]}>
  <Tab value="npm">
    ```bash title="Terminal"
    npm install tiime-sdk
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash title="Terminal"
    pnpm add tiime-sdk
    ```
  </Tab>
</Tabs>

## Exemple rapide

```ts title="app.ts"
import { TiimeClient } from "tiime-sdk";

const client = new TiimeClient({ companyId: 12345 });

// Factures payées
const invoices = await client.invoices.list({ status: "paid" });

// Soldes bancaires
const balances = await client.bankAccounts.balance();

// Info utilisateur
const me = await client.users.me();
```

## Authentification

Le SDK peut s'utiliser de manière autonome, sans le CLI. Plusieurs modes d'auth sont supportés (par ordre de priorité) :

<Tabs items={["Env vars", "Options", "CLI"]}>
  <Tab value="Env vars">
    ```bash title="Terminal"
    export TIIME_EMAIL=vous@example.com
    export TIIME_PASSWORD=votre-mot-de-passe
    export TIIME_COMPANY_ID=12345
    ```

    ```ts title="app.ts"
    const client = new TiimeClient(); // tout est résolu automatiquement
    ```
  </Tab>
  <Tab value="Options">
    ```ts title="app.ts"
    const client = new TiimeClient({
      email: "vous@example.com",
      password: "votre-mot-de-passe",
      companyId: 12345,
    });
    ```
  </Tab>
  <Tab value="CLI">
    ```bash title="Terminal"
    npm install -g tiime-cli
    tiime auth login
    tiime company use --id 12345
    ```

    ```ts title="app.ts"
    const client = new TiimeClient(); // utilise les tokens stockés par le CLI
    ```
  </Tab>
</Tabs>

<Callout type="info" title="Token direct">
  Vous pouvez aussi passer un access token via `TIIME_ACCESS_TOKEN` ou l'option `tokens: { access_token, expires_at }`.
</Callout>

## Ressources disponibles

| Ressource | Accès | Méthodes | Description |
|-----------|-------|----------|-------------|
| Factures | `client.invoices` | 8 | CRUD + envoi + PDF + duplication |
| Devis | `client.quotations` | 5 | CRUD + envoi + PDF |
| Clients | `client.clients` | 4 | CRUD + recherche |
| Comptes bancaires | `client.bankAccounts` | 3 | Liste + détails + soldes |
| Transactions | `client.bankTransactions` | 6 | Liste + pagination auto + imputation |
| Notes de frais | `client.expenseReports` | 3 | CRUD |
| Documents | `client.documents` | 5 | Liste + catégories + upload + download |
| Labels | `client.labels` | 3 | Personnalisés + standards + tags |
| Utilisateurs | `client.users` | 3 | Profil + infos légales + settings |
| Entreprise | `client.company` | 4 | Détails + config + dashboard |
| Entreprises | `client.listCompanies()` | 1 | Liste des entreprises du compte |
